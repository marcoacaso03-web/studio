# **App Name**: Squadra+

## Core Features:

- Project setup and base configuration: Initialize a new Next project, Italian locale, and folder structure.
- Navigation Setup: Implement app navigation using go_router, including main routes, match detail, and category screens.
- Design System: Develop a basic design system including a reusable gradient header widget, match/player cards, a green FAB, consistent spacing (16/24), and large typography for titles.
- Match Calendar (Mock): Implement a mock match calendar feature with CRUD operations (add, edit, delete) for matches stored in memory.
- Team Members (Mock): Implement a mock team members feature with CRUD operations for players stored in memory.
- Match Detail (Mock): Implement a mock match detail screen with tabs for match info, player attendance, statistics, and lineup selection.
- Database Integration with Drift: Set up Drift with SQLite and implement database schema for teams, players, matches, attendance, player match stats, and lineup.

## Style Guidelines:

- Primary color: Deep blue (#1A237E) to convey reliability and professionalism, aligning with the serious nature of sports management.
- Background color: Light gray (#F5F5F5), a very low saturation version of the primary color. Suitable for a light color scheme.
- Accent color: Yellow-orange (#FF8F00) analogous to the primary, this color adds emphasis, contrasts strongly, and evokes energy and motivation.
- Body and headline font: 'PT Sans' (sans-serif) for a modern and slightly warm feel.
- Use consistent and clear icons to represent different actions and categories (e.g., calendar, members, statistics).
- Employ a clean and structured layout with consistent spacing (16/24) for easy navigation and readability.
- Use subtle animations for transitions and feedback to enhance user experience without being distracting.
